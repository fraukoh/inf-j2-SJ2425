Aufgabe 17:

Erzeuge ein Array zufallszahlen mit 10 Zufallszahlen zwischen 1 und 100. > Hinweis: recherchiere selber, wie man eine Zufallszahl in python erzeugen kann.

Code 
import random
zufallszahlen = []
for ind in range(10):
    zufallszahlen.append(random.randint(1, 100))
print(zufallszahlen)
[2, 93, 98, 90, 59, 25, 70, 20, 33, 44]
